# Features

> What RewindX can do for you — explained in plain language.

---

## 🧠 Smart Memory

RewindX doesn't just store data — it **remembers what matters**.

### Ask Anything

```
"What was I debugging yesterday?"
"Show me all React work this week"
"When did I fix the login bug?"
"What did we discuss in Monday's meeting?"
```

RewindX searches across:
- Screenshots (with AI analysis)
- Git commits
- Terminal commands
- Browser history
- File changes
- Meetings

### Time Travel

Browse your work day as a visual timeline:
- Scrub through screenshots
- See what you were doing at any moment
- One-click to restore your workspace

### Automatic Journals

Every day, RewindX generates a summary:
- What you worked on
- How long you spent
- What you accomplished
- What's next
