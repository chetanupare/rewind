# Contributing to RewindX

Thank you for your interest in contributing to RewindX!

## How to Contribute

### Reporting Issues
- Use GitHub Issues
- Include steps to reproduce
- Include OS and Node.js version

### Pull Requests
1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Add tests if applicable
5. Submit a pull request

## Development Setup

```bash
git clone https://github.com/chetanupare/rewind.git
cd rewind
npm install
npm run build
```

## Code Style
- TypeScript
- ESLint for linting
- Prettier for formatting
