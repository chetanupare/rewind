# RewindX Roadmap

> **Vision:** Transform from an event logger into a true digital twin that knows, predicts, and assists.

---

## Phase 1 — Cognitive Core ⭐⭐⭐⭐⭐
**Status:** ✅ Implemented

Create a single orchestrator that everything plugs into.

```
Brain
├── Perception    ← Collectors
├── Memory        ← Knowledge Graph, Episodes, Working Memory
├── Planning      ← Goal Engine
├── Reasoning     ← Evidence-based answers
├── Reflection    ← Nightly thinking
└── Execution     ← Actions, notifications
```

**Why:** Features should be services used by the Brain, not independent modules.

---

## Phase 2 — Goal Engine ⭐⭐⭐⭐⭐
**Status:** 🔨 In Progress

Every episode should belong to a goal.

```
Current App: VS Code
Goal: Implement OCR
Status: 75%
Started: Yesterday
Blocked: Tesseract
Dependencies: Vision Pipeline
Expected Next: Testing
```

**Impact:** RewindX feels alive — it knows what you're trying to accomplish.

---

## Phase 3 — Active Memory
**Status:** 📋 Planned

Searching becomes the last resort.

```
User asks → Working Memory → Goal Memory → Episode Memory
         → Knowledge Graph → Reasoning → Answer
```

**Why:** The AI should already know context before searching.

---
